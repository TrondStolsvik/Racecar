The Code for Racecar. 

              
                function getString() {
                  //Clear the html
                  document.getElementById("results").innerHTML = "";
                
                  //Add "invisible" class to the results id to hide it before the start
                  document.getElementById("results").classList.add("invisible");
                
                  //Declare a variable palInput and store the input from the dom
                  let palInput = document.getElementById("palindromeInput").value;
                
                  //Call stringReverse with the parameter palInput and store it in a new variable called revInput
                  let revInput = stringReverse(palInput);
                
                  //Call palCheck and pass it the variables palInput and revInput and store it's result in a variable called newString
                  let newString = palCheck(palInput, revInput);
                
                  //Call palDisplay and pass it the variable newString so that we can display our result
                  palDisplay(newString);
                }
                
                function stringReverse(palInput) {
                  //Declare a new empty array called revString to store the reversed string
                  let revString = [];
                
                  //For-loop through the string to reverse it
                  for (let i = palInput.length - 1; i >= 0; i--) {
                    //Concatenate the indexes from the string palInput into revString, starting from the end of the string
                    revString += palInput[i];
                  }
                
                  //Return revString
                  return revString;
                }
                
                function palCheck(palInput, revInput) {
                  //Store the data from the parameter palInput in a variable called palInutCheck
                  //We also remove spaces and special characters using .replace
                  let palInputCheck = palInput.replace(/[^a-zA-Z0-9]/g, "");
                
                  //Store the data from the parameter revInput in a variable called revInputCheck
                  //We also remove spaces and special characters using .replace
                  let revInputCheck = revInput.replace(/[^a-zA-Z0-9]/g, "");
                
                  //Use an if-statement to check if palInputCheck is equal to revInputCheck and force lower case for the check only
                  if (palInputCheck.toLowerCase() == revInputCheck.toLowerCase()) {
                    //If they are a match, concatenate the input to a string in a new variable called newString and return newString
                    let newString = `Great! Your string is a palindrome! It was reversed as: ${revInputCheck}`;
                
                    return newString;
                  }
                  //return palInputCheck
                  else {
                    //If they are not a match, concatenate the input to a different string in a new variable called newString and return newString
                    let newString = `Oh no! Your string is not a palindrome. It was reversed as: ${revInputCheck}`;
                
                    return newString;
                  }
                }
                
                function palDisplay(readyString) {
                  //Remove the "invisible" class from the results id
                  document.getElementById("results").classList.remove("invisible");
                
                  //Inject the result on the page
                  document.getElementById("results").innerHTML = readyString;
                }

The Code is structured in four functions.

getString()

The function getString is responsible for clearing the HTML when we start. It also adds the class "invisible" to the results ID to make sure the results are hidden until we're ready to display.

Next we collect the value from the input and store it in the variable palInput. We pass that variable to the function stringReverse() and store it's result in the variable revInput.

Next we pass the variable revInput to the function palCheck() and we store those results in the variable newString, which we finally pass to the function palDisplay() so that we can show our result.

stringReverse()

The function stringReverse() is responsible for reversing our string so that we can check it against the original string and determine if it's a palindrome or not.

First we declare an empty array to hold our reversed string, we call it revString.

Next we use a for-loop to reverse through the string one iteration at a time. For each index of the string/array, we concatenate the result to the array we declared earlier, revString.

Once we're done with the loop, we return revString.

palCheck()

The function palCheck() is responsible for checking the original string, palInput, against our reversed string, revInput. We declare a variable for each, palInputCheck and revInputCheck where we store the inputs, but make note that we also make sure to remove special characters and spaces by doing a .replace on our inputs, replacing everything but a-z (lower and upper case) and 0-9 with an empty string "".

Next we use an if-statement to check whether or not our inputs are equal to one another. We also force the check to be in lower case, so that case sensitivity is not an issue.

If the strings are a match, we declare a new variable called newString and store a string within it notifying the user that they did indeed enter a palindrome. We pass the reversed input into the string, using a template literal.

If the strings are not a match, we have the else-portion which essentially does the same, except the string itself will notify the user that it was in fact not a palindrome.

palDisplay()

The function palDisplay() takes the parameter readyString which is the final string ready for display.

It removes the "invisible" class from the "results" ID and then injects readyString into the HTML at the "results" ID.
